# VotAppEdu 🗳️

> **Sistema Integral de Votaciones para Centros Educativos** > Proyecto Final de Ciclo - IES Joan Coromines

VotAppEdu es una solución digital diseñada para modernizar la toma de decisiones en el entorno escolar. Desde la elección del Consejo Escolar hasta consultas rápidas de aula, el sistema garantiza un proceso seguro, transparente y accesible para todos.

## 🌟 Características Principales

### 1. Versatilidad de Consultas
El sistema no se limita a un solo tipo de elección. Permite configurar:
* **Selección única o múltiple:** Adaptable según la normativa de cada votación.
* **Votaciones por categorías:** Diferentes opciones para alumnado, profesorado y familias.
* **Ámbito total:** Útil para delegados, encuestas de satisfacción o presupuestos participativos.

### 2. Seguridad y Privacidad
* **Identificación por DNI:** Garantiza que cada persona vote solo una vez.
* **Voto Encriptado:** Los votos se almacenan de forma segura en la base de datos MySQL.
* **Anonimato verificado:** El sistema genera un identificador único para que el usuario pueda comprobar que su voto ha sido contado sin revelar su elección.

### 3. Visualización en Tiempo Real
* **Gráficos Dinámicos:** Seguimiento de la participación y resultados mediante Chart.js.
* **Panel de Administración:** Gestión total de usuarios, apertura y cierre de urnas digitales.
